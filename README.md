Таблицы Шульте - классическая игра-тренажёр на развитие внимания и периферического зрения.
Задача игры - нажимать на цифры по возрастанию, пока они не закончатся. ОЧЕНЬ ВАЖНО смотреть в центр таблицы, а не на цифры.
В папке находятся два основных файла - игра в в формате exe (можно играть без установки питона) и игра в формате py, чтобы играть в среде питона.
